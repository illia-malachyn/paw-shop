2. HandleBatch method — POST /api/orders/batch:
```go
// HandleBatch — POST /api/orders/batch
// Приймає order_ids та action ("confirm"|"reject"), застосовує MacroCommand.
func (h *OrderHandler) HandleBatch(w http.ResponseWriter, r *http.Request) {
```
- Check method is POST, return 405 otherwise
- Decode JSON body with `var req struct { OrderIDs []string \`json:"order_ids"\`; Action string \`json:"action"\` }`
- Validate action is "confirm" or "reject", return 400 otherwise
- Look up each order by ID from h.orders; if any ID not found, return 400 with `"order not found: {id}"`
- Build []order.OrderCommand — for each order, create ConfirmOrderCommand or RejectOrderCommand based on action
- Create MacroCommand via order.NewMacroCommand(commands)
- Call macro.Execute(); if error, return 400 with error message
- Return 200 JSON: `{"processed": len(req.OrderIDs), "action": req.Action}`
