```go
func (h *OrderHandler) HandleStatus(w http.ResponseWriter, r *http.Request) {
```
- Check method is PATCH, return 405 otherwise
- Extract order ID from URL path: strip "/api/orders/" prefix and "/status" suffix
- Parse JSON body: `var req struct { Action string `json:"action"` }`
- Look up order via h.collection.GetByID(id), return 404 if not found
- If action == "next", call order.Next(); if action == "cancel", call order.Cancel(); otherwise return 400 "invalid action"
- If Next()/Cancel() returns error, return 400 with the error message
- On success, return 200 JSON: `{"id": order.ID, "status": order.GetState().Name()}`

6. Add HandleListOrders method — GET /api/orders:
```go
func (h *OrderHandler) HandleListOrders(w http.ResponseWriter, r *http.Request) {
```
- Check method is GET, return 405 otherwise
- Check query param: status := r.URL.Query().Get("status")
- If status is not empty, use h.collection.CreateFilteredIterator(status); otherwise use h.collection.CreateIterator()
- Iterate via HasNext()/Next() to collect orders into a slice
- Return 200 JSON array of orders

7. Add HandleCreateOrder method — POST /api/orders:
```go
func (h *OrderHandler) HandleCreateOrder(w http.ResponseWriter, r *http.Request) {
```
- Check method is POST, return 405 otherwise
- Parse JSON body:
```go
var req struct {
    Items   []string `json:"items"`
    Address string   `json:"address"`
    Amount  float64  `json:"amount"`
}
```
- Create an order.OrderRequest from the parsed body
- Run through h.validator.Validate(&orderReq); if error, return 400 with error message
- Generate order ID: fmt.Sprintf("order-%d", h.collection.Count()+1)
- Create order via order.NewOrder(id, req.Items)
- Add to collection via h.collection.Add(newOrder)
- Return 201 JSON: `{"id": newOrder.ID, "status": newOrder.GetState().Name(), "items": newOrder.Items}`

8. Add HandleOrders method — router for /api/orders path:
```go
func (h *OrderHandler) HandleOrders(w http.ResponseWriter, r *http.Request) {
```

- If path matches "/api/orders/*/status" and method is PATCH, delegate to HandleStatus
- Otherwise return 405

- The HandleOrders method internally routes based on path and method
