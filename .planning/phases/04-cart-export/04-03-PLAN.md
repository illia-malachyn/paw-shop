    Define CartHandler struct:
    ```go
    type CartHandler struct {
        cart    *cart.Cart
        history *cart.CartHistory
    }
    ```

    Constructor `NewCartHandler() *CartHandler` — initializes with empty Cart and empty CartHistory.

    Implement handler methods following existing patterns (see orders.go for style):

    **HandleCart(w, r)** — route multiplexer for `/api/cart` and `/api/cart/*`:
    - Parse URL path to determine sub-route.
    - `/api/cart` with GET → return current cart state as JSON.
    - `/api/cart/add` with POST → handle add.
    - `/api/cart/remove` with POST → handle remove.
    - `/api/cart/undo` with POST → handle undo.
    - `/api/cart/export` with GET → handle export.
    - Wrong method → 405 Method Not Allowed.

    **Add logic** (POST /api/cart/add):
    - Decode JSON body into `cart.CartItem` (fields: product_id, name, price, quantity).
    - Validate: product_id and name must be non-empty, price > 0, quantity > 0. Return 400 if invalid.
    - Save current state to history BEFORE mutation: `h.history.Push(h.cart.Save())`.
    - Call `h.cart.AddItem(item)`.
    - Return 200 with JSON `{"status": "added", "cart": h.cart.Items}`.

    **Remove logic** (POST /api/cart/remove):
    - Decode JSON body: `{"product_id": "..."}`.
    - Save state to history BEFORE mutation.
    - Call `h.cart.RemoveItem(productID)`. If error → return 404 with error message.
    - Return 200 with JSON `{"status": "removed", "cart": h.cart.Items}`.

    **Undo logic** (POST /api/cart/undo):
    - Pop from history. If empty (ok==false) → return 400 `{"error": "nothing to undo"}`.
    - Call `h.cart.Restore(memento)`.
    - Return 200 with JSON `{"status": "undone", "cart": h.cart.Items}`.

    **Get cart** (GET /api/cart):
    - Return 200 with JSON of cart.Items. If nil, return empty array `[]`.

    **Export logic** (GET /api/cart/export?format=json|text):
    - Read `format` query parameter.
    - If "json" → create `&export.JSONExportVisitor{}`, call `export.ExportCart(*h.cart, v)`.
    - If "text" → create `&export.TextReceiptVisitor{}`, call `export.ExportCart(*h.cart, v)`.
    - If missing or unknown format → return 400 `{"error": "format must be json or text"}`.
    - For JSON format: set Content-Type "application/json", write result directly (it's already JSON).
    - For text format: set Content-Type "text/plain", write result.

    **Update cmd/server/main.go:**
    - Add `cartHandler := handler.NewCartHandler()` after chatHandler initialization.
    - Register route: `http.HandleFunc("/api/cart/", cartHandler.HandleCart)` — trailing slash catches all sub-paths.
    - Also register `http.HandleFunc("/api/cart", cartHandler.HandleCart)` for exact `/api/cart` path (GET cart).

    Follow existing conventions: Ukrainian doc comments on exported types/methods, method check early return pattern, anonymous struct for request bodies.
