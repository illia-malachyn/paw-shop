`HandleSend(w http.ResponseWriter, r *http.Request)` -- POST /api/chat/send:
1. Check POST method.
2. Decode JSON body: `{"from": "customer1", "to": "manager1", "message": "Hello"}`.
3. Validate from, to, message are non-empty -> 400 if missing.
4. Call `h.mediator.SendMessage(from, to, message)`.
5. Return 200 with `{"status": "sent", "from": from, "to": to}`.

`HandleHistory(w http.ResponseWriter, r *http.Request)` -- GET /api/chat/history?participant=:
1. Check GET method.
2. Get `participant` query param. If empty, return 400.
3. Call `h.mediator.GetHistory(participant)`.
4. Return 200 with JSON array of Message objects.

`HandleChat(w http.ResponseWriter, r *http.Request)` -- router method (like HandleOrders):
- `/api/chat/send` + POST -> HandleSend
- `/api/chat/history` + GET -> HandleHistory
- otherwise -> 405

Import: `"github.com/illia-malachyn/paw-shop/internal/chat"`.
